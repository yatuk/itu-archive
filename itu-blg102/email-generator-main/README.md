Email Generator
```
The program will generate three email addresses based on the provided information using different strategies:
Strategy 1: Initial of name + surname + last 2 digits of year
Strategy 2: First two characters of the name + first two characters of the surname + 4-digit random number
Strategy 3: Surname + Name + Year
```
Example Output : 
```
***********************************
Welcome to Email Generator
Please Enter Some Information
***********************************
Please enter your name :
John
Please enter your surname :
Doe
Please enter your birth year :
1985

Strategy 1: JDoe85@gmail.com
Strategy 2: JoDo4315@gmail.com
Strategy 3: DoeJohn1985@gmail.com
```
